. Авторизация / регистрация

2. Основной интерфейс пользователя
    Отображается список с заголовком "Мои словари".
  К списку привязываем функционал :
      1. Создать словарь
    2. Изменить словарь
    3. Выбрать словарь (выбор осуществляется из словарей, созданных другими пользователями, не отмеченных как частные)
    4. Настройка режима изучения
    5. Запустить тест

  В списке у каждого словаря отображается :
        1. Тема
       2. Количество слов
    3. Количество изученных слов (вычисляемое)
    4. Дата и время последнего теста.

3. Работа со словарём
    Отображается тема словаря и список слов.
    К списку слов привязываем функционал :
        1. Добавить слово
        2. Удалить слово
        3. Изменить слово (Вот тут не понятно как быть, ибо список слов общий на всех и может быть ситуация, когда один пользователь подкорректирует слова,
                       а эти слова использует другой в своём словаре. Так можно подлянки друг другу делать. Надо подумать!!!!)
    Общий функционал словаря : Сохранить, Отменить.

4. Работа со словом.
    Два поля и функционал : Сохранить, Отменить

5. Настройка режима изучения словаря.
    Отображаются параметры настройки. Сохранить, Отменить.

6. Режим Теста
    Back формирует список слов для изучения из словаря. Размер списка согласно настроек. Подряд друг за другом пользователю отображаются слова.
    Пользователь видит слово на английском. Вводит перевод на русском. Идёт сравнение. Если корректно, то результат сохраняется.
